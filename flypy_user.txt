# 小鹤音形用户词库（和系统词条重码时居后）
# 该文件以 UTF-8 编码保存
# 编码各字段以制表符（Tab）分隔
# 编码格式：字词 Tab编码 Tab权重（权重决定重码的次序，权重大则居前，用户词库本身无重码则无需权重）
#
#    === 强调一下 === 
#
#   词条和编码之间的不是空格，而是 Tab 符，如果你不知道，就复制下面编码中的空白处
#   或者按住键盘 G 键，切换到功能键盘，使用上面的 Tab 键
#
#    ================
#
# 如两个用户词重码，则需加权重排序，权重大者居前，如下：
#
# 同文	tw	90
# 提问	tw	30
#
# 如只加一个用户词，无需权重（如想置于系统词 “台湾” 前，请把词条放到 flypy_top 文件内），如下：
#
# 同文	tw
#
#    ================
#
